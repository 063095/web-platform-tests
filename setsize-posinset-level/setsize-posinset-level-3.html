Description

A div element with the role='tree' has three child div elements with role='treeitem'. Each of the child div elements contains a div element with the role of tree with two child div elements with the role='treeitem'. When focus is on the fifth div element with a role of "treeitem" in document order will be represented in the accessibility tree using the following table:

http://www.w3.org/TR/2012/WD-wai-aria-implementation-20120110/#mapping_state-property_table

Accessibility API Information

  • set size: 2
  • position in set: 1
  • level of set in the hierarchy: 2
